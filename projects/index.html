Nonlinear Peer-Aware Cost Estimation

Aug 2025 - Ongoing | GHOST Lab, ASU

02 show project details expand

Mobile robot navigation using a nonlinear cost function with motion-capture calibration, EKF filtering, and model predictive control.

IJRR manuscript submission is currently in progress for this project.

tech: ROS, C++, Motion Capture, EKF, MPC

  • Programmed a mobile robot to move based on a nonlinear peer-aware cost function.
  • Calibrated motion capture to estimate real-time positions of rigid bodies.
  • Developed ROS nodes in C++ for real-time control and feedback collection.
  • Used Extended Kalman Filter to reduce motion-capture noise.

LQR Control for Motor Position Control

04 show project details expand

tech: SIMULINK, Arduino, ROS2, Gazebo, LQR, State Feedback, IIR Filtering
